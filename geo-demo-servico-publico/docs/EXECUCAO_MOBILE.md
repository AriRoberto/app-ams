# Execução Mobile

## Pré-requisitos
- Node.js
- Android Studio (Android)
- Xcode (iOS/macOS)

## Rodar Android
```bash
cd geo-demo-servico-publico/mobile
npm install
EXPO_PUBLIC_API_URL=http://10.0.2.2:3340/api npm run android
```

## Rodar iOS
```bash
EXPO_PUBLIC_API_URL=http://localhost:3340/api npm run ios
```
