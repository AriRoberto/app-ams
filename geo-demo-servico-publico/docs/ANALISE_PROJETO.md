# Análise do Projeto

## Escopo oficial
A demo oficial do repositório é exclusivamente `geo-demo-servico-publico`, composta por backend, frontend e mobile.

## Pontos centrais
- Geolocalização territorial com IBGE + Nominatim.
- Registro de ocorrência urbana estruturada.
- Prévia formal de e-mail para encaminhamento institucional.

## Fluxo funcional
1. Carregar geolocalização oficial.
2. Exibir cidade, coordenadas e bounding box em mapa.
3. Capturar ocorrência e validar campos.
4. Gerar saída institucional pronta para envio.
